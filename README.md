Instructions to Run Scene in Unity Hub:

To load the project with everything

Ensure the Unity version is 6000.3.11f1 or later

1.Open unity project in unity hub by adding the project
2.Wait for the project to fully load
3.Load scene in Assets folder called "Input2_restaurant.unity"
4.Press play button while headset is on and linked to PC or via desktop in the Meta Development Environment by having the Meta Horizon Link application open with the Meta Quest Device setup in "Devices".
5.Interact with the canvas UI menus to spawn food/drink Prefabs and drag them to checkout zones to add to carts.
6.Clear contents of carts with the Clear Cart button or checkout with the Checkout button.

When finished click the stop button on the PC to close the scene or take a checkout bag regardless of contents and walk towards the exit door.
